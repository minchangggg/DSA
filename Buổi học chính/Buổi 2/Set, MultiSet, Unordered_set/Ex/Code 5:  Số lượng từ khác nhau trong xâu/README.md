        }
        
`Input Format`

Xâu kí tự S.

`Constraints`

1 ≤ len(S) ≤ 10^6;

`Output Format`

Dòng đầu tiên in ra số lượng từ khác nhau trong xâu. 

Dòng thứ 2 in ra từ có thứ tự từ điển nhỏ nhất và lớn nhất trong xâu.

`Sample Input 0`

ngo ngoc an nguyen an ngoc ngoc

`Sample Output 0`

4

an nguyen
