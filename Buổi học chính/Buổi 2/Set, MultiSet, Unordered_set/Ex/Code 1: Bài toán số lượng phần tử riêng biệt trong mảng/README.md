## Đề bài : Cho mảng A[] gồm N (1 ≤ N ≤ 106)  phần tử, hãy đếm xem trong mảng có bao nhiêu giá trị khác nhau ? 
- Giải thuật : Đưa tất cả các phần tử trong mảng vào set và in ra kích thước của set
- Độ phức tạp : O(NlogN)
