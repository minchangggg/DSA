`Input`

2 số nguyên không âm a, b (0 <= a <= b <= 10^9, b-a <= 10^5)

`Output`

In ra các số nguyên tố trong đoạn từ a tới b (Chú ý lấy cả 2 cận)

`Ví dụ`

        Input        Output
        4 20         5 7 11 13 17 19
        1 5          2 3 5
